# ydydj-sniper 旧版本 爆破版本

暴力破解查询指定运动员的运动员技术等级证书的编号

## 引子

曾几何时，查询运动员技术等级证书必须同时提供运动员姓名和证书编号了。为防止大家忘记编号，丢了证书，特写出此爆破编号工具。

请牢记官方查询地址：

其一 https://ydydj.univsport.com/level/

其二 https://zwfw.sport.gov.cn/level.html

## 使用指北

保证在电脑上安装有python

在terminal中进入项目目录，运行以下命令安装依赖：

```bash
pip install -r requirements.txt
```

仍在项目目录下，运行以下命令运行主程序：

```bash
python sniper.py
```

随后即可在按照提示输入：运动员姓名和编号可能的范围：

```
PS C:\...\ydydg-cli> python .\sniper.py
请输入运动员姓名: 刘翔
请输入起始证书号 (如 20210000): 20080000
请输入结束证书号 (如 20210099): 20089999
```

等待片刻后，输出结果示例：

```
--------------------------------------------------
查询进度: 100%|██████| 10000/10000 [00:00<01:00, 116.91it/s]
==================================================       
✓ 找到有效证书编号: 1 个
  已保存到: certificates.txt
==================================================       
查询到的等级证书编号:
  1. 证书编号: 2008xxxx | 刘翔 - 国际级运动健将 - 田径
```

特别提示，在2015年后获得证书的，编号常常为八位，前4位为年份，后4位为数字，如证书颁发在2021年，可以使用起始证书号20210000，结束证书号20210099做查询。

另外提示，若在次年申请证书，编号常常为九位，前4位为申请证书当年年份，后5位位纯数字，但数字通常较小。例如2021年比赛达级，2022年申请证书，则可以用起始证书号202200000，结束证书号202200999做查询。

在2015年前获得证书的，编号位数通常更长，有时有字母，暂未发觉规律。

## 技术小注

使用python。并发默认每次100个请求，1分钟约可执行1000号查询。查询到的证书编号记录在项目目录下`cetificates.txt`文件中。

每次查询日志写在logs文件夹内，命名规则：`运动员姓名_查询时间_查询范围_范围大小`。所有查询记录在`history.json`缓存文件中，新的查询会跳过有历史缓存的查询。
